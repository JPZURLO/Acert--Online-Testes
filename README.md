# Projeto de Controle de Obras Públicas

Este projeto é uma **plataforma digital colaborativa** desenvolvida para **gestão de obras públicas**. O sistema permite o **acompanhamento em tempo real** de obras em andamento, promovendo maior **transparência** e **eficiência** na gestão pública. A plataforma centraliza as informações de obras, incluindo dados sobre prazos, orçamento, problemas, soluções e impacto, além de facilitar a comunicação entre **engenheiros**, **gestores** e **cidadãos**.

## Funcionalidades

- **Dashboard de obras**: Interface para engenheiros e gestores visualizarem o andamento das obras em tempo real.
- **Mapa interativo**: Visualização de obras por região e estado, com detalhes sobre o número de obras em cada local.
- **Tela de detalhes da obra**: Informações detalhadas sobre cada obra, incluindo status, responsáveis e orçamento.
- **Relatórios e análises**: Geração de relatórios sobre obras em andamento, atrasadas e concluídas.
- **Tela de feedback e comunicação**: Sistema para engenheiros, gestores e cidadãos interagirem, oferecendo feedback sobre as obras.

## Tecnologias Utilizadas

- **Frontend**:
  - HTML
  - CSS
  - JavaScript
- **Backend**:
  - Node.js
  - Express
  - JWT (para autenticação)
  - PostgreSQL (banco de dados)
- **Hospedagem**:
  - Heroku (para a hospedagem da aplicação)
